Current GIS solutions

Currently the Isle of Wight Council uses a number of proprietary tools for common Geographical Information System related tasks such as data visualization and mapping. The current tools require a large amount of financial resources. The IoW Council challanges the current setup and investigates the possibilties to improve the situation. The Open-source initative

In 2017 a new initiative was accepted that intends to move the current infrastructure to a free and open-source technology based solution. This task requires a thorough investigation of available GIS tools.
